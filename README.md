# 🌱 Virtual Plant Health Care System

## 📌 Project Description

The **Virtual Plant Health Care System** is a full-stack web application designed to assist users in managing and monitoring plant care activities efficiently. It provides an interactive platform where users can track plant health, receive reminders, and access plant-related information.

The system addresses real-world challenges such as irregular plant maintenance, lack of awareness, and inefficient tracking by offering a structured and user-friendly digital solution.

---

## 🎯 Objectives

* To develop an interactive web-based plant care system
* To simplify plant maintenance for users
* To provide reminders for watering and plant care
* To demonstrate full-stack web development concepts

## 🧩 System Architecture

The system follows a **three-layer architecture**:

### 1. Frontend Layer

* Built using React and TypeScript
* Handles UI rendering and user interaction
* Uses component-based structure

### 2. Backend Layer

* Implemented using Convex
* Handles authentication, API requests, and business logic
* Ensures real-time data synchronization

### 3. Database Layer

* Stores user and plant data
* Maintains data consistency and persistence

---

## 🔄 Working Flow

1. User interacts with the frontend interface
2. Frontend sends request to backend (Convex)
3. Backend processes and validates data
4. Data is stored/retrieved from database
5. Response is sent back to frontend
6. UI updates dynamically
